CSS Grid Without Text-Align

Grid Box 1

CSS Grid Layout is a 2-dimensional system in that it uses columns and rows. Like flex, it uses a container as a parent element and the grid items as the element's children.

Grid Box 2

The container is set up with display:grid and the column and rows sizes are set up with grid-template-columns and grid-template rows.

Grid Box 3

The children or grid items can be placed onto the grid layout using grid-column and grid-row.

Grid Box 4

A powerful featur of the grid layout is that columns are fluid and can break into more or less columns as space is available without media queries.

Grid Box 5

The minmax() function is used to define the miniumum and maximum values we want the column to resize to as the screen viewport is adjusted.

Grid Box 6

I used auto fill to fill the rows with as many columns as fitted (boxes 1-6 and boxes 10-15). And, I used auto fit to expand the columns into the available space (boxes 7, 8 and 9).
