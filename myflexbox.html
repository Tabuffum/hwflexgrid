FlexBox Without Text-Align

Flex Box 1

The most important thing I learned about the flex box is that it relies on the parent element, the flex container. The items within the flex container are the child elements.

Flex Box 2

The container drives the flex layout of the items within the container. The flex container expands the items to fill the container's free space or shrink them to prevent overflow.

Flex Box 3

By default, the flex items inside the container will attempt to fit onto one line. I set the flex to wrap so the items wrap onto the next line as needed.

Flex Box 4

There are all kinds of attributes available to control the flow of the flex (the wrap and direction), the alignment of the items within the container.

Flex Box 5

I set the flex flow for Flex Boxes, 7, 8, and 9 to column wrap to illustrate the direction attribute of flex boxes.

Flex Box 6

I set container 3 to flex grow by a factor of 4 so boxes 10, 11, 12 and boxes 13, 14, and 15 would grow to fill their entire containers.
